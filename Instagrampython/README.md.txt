This is a simple python program that gives you the unfollowers list.
First of all you need to login to the instagram in desktop.
Go to settings> Privacy and security> Account data > View account data.

Under the Connections tab> Account folllowing you 
click view all
copy all the list of Accounts following you and paste it to text file and rename it "accountsfme"

Again,
Under the Connections tab> Account you follow
click view all
copy all the list of Accounts you follow and paste it to text file and rename it "accountsifollow"

Then, Execute the python file "findunfollowers.py"
It gives you the UNFOLLOWERS LIST.

***Remember: All of these three files should be in the same folder.
